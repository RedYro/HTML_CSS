C - Direction

"Flexbox" nous permet d'agencer ces éléménts dans le sens que l'on veut. Avec "flex-direction", on peut les positionner verticalement ou les inverser. Il peut prendre les valeurs suivantes:
-> "row" : organisés sur une ligne (par défaut)
-> "column" : organisé sur une colonne
-> "row-reverse" : organisé sur une ligne en ordre inversé
-> "column-reverse" : organisé sur une colonne en ordre inversé

D - Retour à la ligne

Par défaut, les blocs essaient de rester sur la même ligne s'ils n'ont pas la place (ce qui provoque parfois des bugs de design). Si vous voulez, vous pouvez demander à ce que les blocs aillent à la ligne lorsqu'ils n'ont plus de place avec "flex-wrap" qui peut prendre ces valeurs :
-> "nowrap" : pas de retour à la ligne (par défaut)
-> "wrap" : les éléments vont à la ligne lorsqu'il n'y a plus de place
-> "wrap-reverse" : les éléments vont à la ligne lorsqu'il n'y a plus de place dans le sens inverse

E - Alignement

Les éléments sont organisés soit horizontalement (par défaut), soit verticalement. Cela définit, ce que l'on appelle l'axe principal. Il y a aussi un axe secondaire (cross axis) :
-> si vos éléments sont organisés horizontalement, l'axe secondaire est l'axe vertical
-> si vos éléménts sont organisés verticalement, l'axe secondaire est l'axe horizontal

Alignement sur l'axe principal

Afin d'aligner nos éléments sur l'axe principal : "justify-content"
elle peut prendre comme valeur :
-> "flex-start" : aligner au début (par défaut)
->"flex-end" : aligner à la fin
->"center" : aligner au centre
->"space-between" : les éléments sont étirés sur tout l'axe (espace entre les blocs)
->"space-around" : les éléments sont étirés sur tout l'axe (espace sur les extrémités)
->"space-evenly" : les éléments sont répartis uniformément dans le conteneur d'alignement le long de l'axe principal

F - Répartir plusieurs lignes

Si vous avez plusieurs lignes dans votre "flex-box" vous pouvez choisir comment celles-ci seront réparties avec "align-content".
Cette propriété permet de gérer l'alignement des différentes lignes (ou colonnes) les uns par rapport aux autres.
Cette propriété n'aura d'effet que si le conteneur flexible possède plusieurs lignes (si l'axe principal est horizontal) ou plusieurs colonnes (si l'axe est vertical) et que si le "flex-wrap" est présent.
Les valeurs de cette propriété :
->"flex-start" : éléments placés au début
->"flex-end" : éléménts placés à la fin
->"center" : éléments placés au centre
->"space-between" : éléments séparés avec un espace entre eux
->"space-around" : éléments séparés avec un espace entre eux mais aussi au début et à la fin
->"stretch" (par défaut) : éléments étirés pour occuper tout l'espace
->"space-evenly" : éléments répartis uniformément dans le conteneur d'alignement le long de l'axe principal
